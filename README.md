Employee Management API
A simple RESTful API built with Spring Boot to manage employee records. Uses H2 in-memory databasefor storing data.
## Features
- Fetch employee details by email or name
- Create employees with name and email, or name, email, and phone
- Update employee details or phone only
- Delete employee by email
- Follows   RESTful API design principles
## Technology Stack
- Java 17+
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven
